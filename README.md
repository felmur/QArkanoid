# QArkanoid
Simple "Arkanoid" type game, based on QT libraries, with sounds.

This game is written for Linux, but it's probably windows+Qt compatible. I need someone who want try this.


## Compilation & usage

Open a linux bash and type:

$> qmake

$> make

$> ./QArkanoid
